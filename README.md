# Valentine

Проект на **Vue 3** + **TypeScript** + **Vite** + **Vue Router**.

## Запуск

```bash
# Установка зависимостей
npm install

# Режим разработки
npm run dev

# Сборка
npm run build

# Превью собранного приложения
npm run preview
```

После `npm run dev` приложение будет доступно по адресу http://localhost:5173
